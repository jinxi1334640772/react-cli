# zhangjinxi-cli
vue-admin开发脚手架，vue2.6版本，支持项目初始化、自动依赖安装，本地开发、打包功能。

全局安装脚手架：
```
npm i zhangjinxi-cli -g
```
查看版本：
```
zhangjinxi -V
```
项目初始化，新建appName项目：
```
zhangjinxi create appName
```
切换到项目目录下:
```
cd appName
```
本地开发模式（http://localhost:3000/）：
```
npm run serve
```
本地打包：
```
npm run build
```
lint
```
npm run lint
```
